# NAGATO ROOT KIT

Professional Pixel Root Automation Toolkit for macOS.

## Features

- Automatic device detection
- Automatic OTA selection
- Automatic init_boot extraction
- Fresh Setup mode
- Update/Reinstall mode
- Supports:
  - SukiSU
  - KernelSU Next
  - APatch
  - ReSukiSU
- Automatic APK detection
- Automatic module detection
- Automatic config detection
- Automatic patched image detection
- Waits for Android boot
- Automatic app installation
- Automatic cleanup
- Session logging
- Beautiful terminal UI

---

## Supported Devices

- Pixel 6a
- Pixel 7a
- Pixel 7 Pro

More Pixel devices can easily be added.
