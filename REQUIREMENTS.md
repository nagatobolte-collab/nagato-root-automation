# NAGATO ROOT KIT — Requirements

## Supported Operating Systems

- macOS 12+
- macOS 13+
- macOS 14+
- macOS 15+
- Apple Silicon (M1/M2/M3/M4)
- Intel Macs

---

## Required Software

### Android Platform Tools

Install with Homebrew:

brew install --cask android-platform-tools

Verify:

adb version
fastboot --version

---

### Android OTA Extractor

Place the executable in the toolkit root folder:

android-ota-extractor

---

## Supported Devices

- Pixel 6a
- Pixel 7a
- Pixel 7 Pro

---

## Phone Requirements

Enable:

- Developer Options
- USB Debugging

For Fresh Setup:

- OEM Unlocking

## Root Managers

Supported:

- SukiSU
- KernelSU Next
- APatch
- ReSukiSU

## Recommended Modules

- Zygisk Next
- LSPosed
- Play Integrity Fix
- Tricky Store
- Yurikey

---

## Before Rooting

- Backup important data.
- Battery above 50%.
- Use the correct OTA matching your current build.
- Use a USB data cable.
